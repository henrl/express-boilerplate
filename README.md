# Steps to create Node.js boilerplate
1. npm init
2. npx license mit
3. npx gitignore node
4. npx covgen <insert email here>
5. npm i http
6. npm i --save-dev jest ts-jest ts-node typescript
7. npm i dotenv
8. Create a .env file based off the env.sample file and make sure .gitignore includes .env
